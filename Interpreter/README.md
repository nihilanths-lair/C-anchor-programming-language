```
/!\ У вас должен быть установлен компилятор GCC, если планируете из исходного кода собрать исполняемый файл сами
 (в данном мануале как это сделать не оговаривается).

[OS Windows]

Скачиваем файл: gen_prompt.c (1) или prompt.c (2) или prompt.exe (3)

/!\ Первый и второй файл являются не обязательными для работы, но из них можно собрать программу самому.
/!\ Порядок сборки при скачивании файла №1: 1 => 2 => 3, если скачали №2: 2 => 3, при №3 можно сразу запустить его
 (т.к. уже является готовой исполняемой программой).

В зависимости от того, какой файл скачали, переходим к определённым действиям:

 (1) Имя файл gen_prompt.c производим его компиляцию: cc -std=c99 -Wall gen_prompt.c -o prompt.exe
 Получаем файл prompt.exe, который необходим для последующего этапа (будет сгенерирован Си-код)
 Запустим файл prompt.exe и нажмём кнопку `Enter` прямо в консоле
 
 (2) Имея файл prompt.c производим его компиляцию: cc -std=c99 -Wall prompt.c -o prompt.exe
 Получаем файл prompt.exe, который готов для работы, запустить его можно, введя в консоли prompt.exe

 (3) Имея файл prompt.exe можно запустить его для работы, введя в консоли: prompt.exe
```
