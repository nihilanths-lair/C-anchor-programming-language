Архитектура VM (8-bit's)
Объём доступной памяти: 256 байт, через сегментные регистры: 65'536 байт (65 Kb.)


Модель памяти (тип 1)
Инструкции, стек и данные находятся в одной области пространства ; IP == SP == DP
-
Модель памяти (тип 2)
Инструкции и стек находятся в одной области пространства, а данные в другой ; (IP == SP) != DP
-
Модель памяти (тип 3)
Инструкции и данные находятся в одной области пространства, а стек в другой ; (IP == DP) != SP
-
Модель памяти (тип 4)
Инструкции, данные и стек находятся в разных областях пространства ; IP != DP != SP
