Архитектура VM (8-bit's)
Объём доступной памяти: 256 байт, через сегментные регистры: 65'536 байт (65 Kb.)


Модель памяти (тип 1) ; IP == SP == DP
Инструкции, стек и данные находятся в одной области пространства
- -
Модель памяти (тип 2) ; (IP == SP) != DP
Инструкции и стек находятся в одной области пространства, а данные в другой
- -
Модель памяти (тип 3) ; (IP == DP) != SP
Инструкции и данные находятся в одной области пространства, а стек в другой
- -
Модель памяти (тип 4) ; IP != DP != SP
Инструкции, данные и стек находятся в разных областях пространства
- -
Модель памяти (тип 5) ; IP != (DP == SP)
Данные и стек находятся в одной области пространства, инструкции в другой
